Lịch làm việc

Mô tả giao diện: Có một tờ lịch, khi bấm vào ô lịch hiện ra modal thêm ngày nghỉ lễ, ấn gửi dữ liệu post api/calendar/vacations Ô lịch có thể có các màu xanh, đỏ. Xanh là nghỉ thường niên, màu xám là thứ 7, chủ nhật. Đỏ là nghỉ lễ khẩn cấp Role Admin mới cho bấm vào ô lịch, Role thường chỉ được xem lịch Call API để kiểm tra xem ô lịch nào màu xanh, ô lịch nào màu đỏ (api lịch làm việc trong doc, lấy danh sách ngày nghỉ lễ) Cho phép xóa ngày nghỉ dựa cũng bấm vào ô lịch Ngày nghỉ thường niên là 1/5, 30/4, những ngày này cũng là nghỉ có phép Nghỉ khẩn cấp sẽ có vacation-type( là emergency) Yêu cầu: responsive, phần menu sẽ responsive riêng nên nếu tạm vỡ layout không sao Lịch cho phép di chuyển giữa các tháng, dùng thư viện thoải mái. Lịch cho phép tạo ngày làm bù: ví dụ t3 mưa bão, t7 làm bù. Nếu thứ 7 làm bù thì ô lịch sẽ là màu trắng, không phải màu xám nữa Có một nút tạo ngày làm bù Call API bằng ajax (đã cấu hình ajax gửi token) JS viết vào file calendar.js CSS viết vào file calendar.css Trước khi push code, mở claude trong VS code thêm context 3 file calendar.html, calendar.js, calendar.css nhờ nó cải thiện code nhé
